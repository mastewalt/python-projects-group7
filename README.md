## What's Inside?

### 1️⃣ Snack Shop Program
A simple snack shop simulator where you can:
- See a menu of snacks and drinks with prices
- Pick items by number
- Keep adding items until you type "done"
- Get a receipt showing everything you bought and the total price

### 2️⃣ Grocery Cart Program
Practice using dictionaries with this grocery app:
- Start with a list of groceries and their prices
- Add items to your cart by name
- Choose how many of each item you want
- Type "checkout" to see your final bill with subtotals

### 3️⃣ To-Do List Manager
Organize your tasks with this simple manager:
- Add tasks with priority levels (high/medium/low)
- View your current to-do list
- Delete tasks you don't need
- Mark tasks as complete
- See a summary when you exit

### 4️⃣ Movie Ticket Booking
Pretend you're running a movie theater:
- See available movies, showtimes, and prices
- Book tickets for different movies
- Get the total price for each booking
- Book multiple movies in one session
- See your complete booking summary at the end

### 5️⃣ Quiz Game
Test your knowledge with a simple quiz:
- Answer 5-10 questions
- Get your score at the end (like 7/10)
- Learn from your mistakes by seeing the correct answers

## How to Run These Programs

1. Make sure you have Python installed on your computer
2. Download the Python files
3. Open your terminal/command prompt
4. Navigate to where you saved the files
5. Type: `python filename.py` (replace filename with the program you want to run)

Example:
```bash
python question3.py
